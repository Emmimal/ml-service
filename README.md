# ml-service
Machine learning model deployment from notebook to production using FastAPI, Docker, and CI/CD pipelines.

# ML Inference API

Production-grade REST API for serving a scikit-learn GradientBoosting model,
built with FastAPI and Docker.

Part of the **Production ML Engineering** series at
[EmiTechLogic](https://emitechlogic.com/machine-learning-model-deployment/).

```
├── Dockerfile
├── docker-compose.yml
├── requirements.txt
└── requirements-dev.txt
```

## Quickstart (local Python)

```bash
# 1. Install dependencies
pip install -r requirements.txt -r requirements-dev.txt

# 2. Train the model — generates model/artifacts/*.pkl
python -m model.train

# 3. Run the API
uvicorn api.main:app --reload --port 8000

# 4. Run tests
pytest tests/ -v
```

---

## Quickstart (Docker Compose)

```bash
# Train artifacts first (needed before the image build)
pip install -r requirements.txt && python -m model.train

# Build and start all services
docker compose up --build

# Services:
#   API        → http://localhost:8000
#   Docs       → http://localhost:8000/docs
#   Prometheus → http://localhost:9090
#   Grafana    → http://localhost:3000  (admin / admin)
```

## API endpoints

| Method | Path             | Description                          |
|--------|------------------|--------------------------------------|
| GET    | `/health`        | Liveness probe                       |
| GET    | `/ready`         | Readiness probe                      |
| POST   | `/predict`       | Single-sample prediction             |
| POST   | `/predict/batch` | Batch prediction (up to 512 samples) |
| GET    | `/docs`          | Swagger UI                           |
| GET    | `/redoc`         | ReDoc                                |

## Example request

```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "features": [
      17.99, 10.38, 122.8, 1001.0, 0.1184,
      0.2776, 0.3001, 0.1471, 0.2419, 0.07871,
      1.095, 0.9053, 8.589, 153.4, 0.006399,
      0.04904, 0.05373, 0.01587, 0.03003, 0.006193,
      25.38, 17.33, 184.6, 2019.0, 0.1622,
```

## Running in production

The GitHub Actions workflow (`.github/workflows/ml-deploy.yml`) handles:

1. **Test** — lint, unit tests, integration tests
2. **Build** — Docker image pushed to Docker Hub
3. **Deploy staging** — SSH deploy + smoke test
4. **Deploy production** — manual approval gate + SSH deploy
